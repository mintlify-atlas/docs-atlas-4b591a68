## What is Unigram?

Unigram is an open-source Telegram client for Windows that communicates with Telegram servers via [TDLib](https://core.telegram.org/tdlib), Telegram's official cross-platform library. It provides:

- **Native Windows integration** — built with UWP and Fluent Design for a consistent Windows 10/11 experience
- **Full Telegram parity** — chats, channels, bots, stickers, GIFs, polls, voice notes, and more
- **End-to-end encryption** — secret chats with forward secrecy and self-destructing messages
- **Voice and video** — one-on-one calls and group video conferences powered by WebRTC
- **Multi-account support** — switch between multiple Telegram accounts in one window
- **Telegram Premium** — 4 GB file uploads, custom emoji, voice-to-text, and all Premium features

## Getting started

<Steps>
  <Step title="Install Unigram">
    Get Unigram from the [Microsoft Store](https://apps.microsoft.com/detail/9n97zckpd60q) or download the latest `.msixbundle` from [GitHub Releases](https://github.com/UnigramDev/Unigram/releases).
  </Step>
  <Step title="Sign in">
    Open Unigram, enter your phone number, and verify with the code sent to your existing Telegram app or SMS.
  </Step>
  <Step title="Set up your profile">
    Add a profile photo, update your bio, and configure notification preferences to get the experience you want.
  </Step>
  <Step title="Start chatting">
    Search for contacts, join channels, create group chats, or start a secret chat — everything is accessible from the main chat list.
  </Step>
</Steps>
