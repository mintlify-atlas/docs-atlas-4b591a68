---
title: "Keyboard shortcuts"
description: "A complete reference of all keyboard shortcuts in Unigram, organized by category."
---

Unigram supports a comprehensive set of keyboard shortcuts that let you navigate chats, format messages, switch folders, manage calls, and control voice recordings without reaching for the mouse.

You can view and customize shortcuts inside Unigram at **Settings** > **Shortcuts**.

## Navigation

Move between chats and go back and forward in the navigation stack.

| Action | Shortcut |
|--------|----------|
| Next chat | `Ctrl+Tab` |
| Next chat | `Alt+Up` |
| Next chat | `Ctrl+Page Up` |
| Previous chat | `Ctrl+Shift+Tab` |
| Previous chat | `Alt+Down` |
| Previous chat | `Ctrl+Page Down` |
| Go back | `Esc` |
| Go back | `Alt+Left` |
| Go forward | `Alt+Right` |
| Saved messages | `Ctrl+0` |

## Search

| Action | Shortcut |
|--------|----------|
| Search chats | `Ctrl+E` |
| Search messages in current chat | `Ctrl+F` |

## Message actions

Act on messages in the compose area or the message list.

| Action | Shortcut |
|--------|----------|
| Edit last sent message | `Up` |
| Reply to last message | `Ctrl+Up` |
| Close current window | `Ctrl+W` |
| Close Unigram (main window only) | `Ctrl+Q` |
| Change status | `Ctrl+Shift+Y` |

## Text formatting

These shortcuts apply to selected text in the message compose area.

| Action | Shortcut |
|--------|----------|
| Bold | `Ctrl+B` |
| Italic | `Ctrl+I` |
| Create link | `Ctrl+K` |
| Strikethrough | `Ctrl+Shift+X` |
| Monospace | `Ctrl+Shift+M` |
| Spoiler | `Ctrl+Shift+P` |
| Plain text (remove formatting) | `Ctrl+Shift+N` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |
| Cut | `Ctrl+X` |
| Copy | `Ctrl+C` |
| Paste | `Ctrl+V` |
| Select all | `Ctrl+A` |

## Folder switching

Switch between chat folders directly from the keyboard. Folders are numbered in the order they appear in the sidebar.

| Action | Shortcut |
|--------|----------|
| All chats (first folder) | `Ctrl+1` |
| Second folder | `Ctrl+2` |
| Third folder | `Ctrl+3` |
| Fourth folder | `Ctrl+4` |
| Fifth folder | `Ctrl+5` |
| Sixth folder | `Ctrl+6` |
| Seventh folder | `Ctrl+7` |
| Eighth folder | `Ctrl+8` |
| Archive | `Ctrl+9` |

<Note>
  `Ctrl+1` always opens **All chats**. `Ctrl+9` always opens the **Archive**. Folders 2 through 8 correspond to the custom folders you have created, in sidebar order.
</Note>

## Voice note recording

| Action | Shortcut |
|--------|----------|
| Start recording | `Ctrl+R` |
| Send recorded note | `Ctrl+R` (press again while recording) |
| Stop recording (discard) | `Ctrl+D` |
| Pause recording | `Space` (while recording) |
| Pause recording | `Ctrl+P` |

<Tip>
  Press `Ctrl+R` once to start recording a voice note, then press `Ctrl+R` again to send it immediately. To discard without sending, press `Ctrl+D`.
</Tip>

## Calls

| Action | Shortcut |
|--------|----------|
| Accept incoming call | `Ctrl+Home` |
| Reject incoming call | `Ctrl+End` |
| Toggle camera | `Ctrl+Page Up` |
| Toggle microphone | `Ctrl+Page Down` |

<Note>
  `Ctrl+Page Up` and `Ctrl+Page Down` toggle camera and microphone **during an active call**. Outside of a call, those same shortcuts move to the next or previous chat.
</Note>
